This is the resume dataset used for resume block classification.
The format of each dataset is a txt file, the content of one resume is devided by a separator:"------------------------------------", 
and each block in the same resume is divided by a separator:"###".

"data_art" means that this dataset is the art domain, "data_engine" means that this dataset is the engine domain,
"data_it" means that this dataset is the IT domain.
